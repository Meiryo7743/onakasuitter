使い方

上の「ツイートする」ボタンを押下すると、新しいタブまたは Twitter アプリが開かれ、下の画像のような状態に遷移します。あとはツイートボタンを押して、お腹がペコペコなことをアピールしましょう！

なお、生成されるツイートに付随の URL（画像）は、@naname_eis 様のものです。利用を快諾してくださったこと、心より感謝申し上げます。

onakasuitter ではホスティングサービスとして GitHub Pages を利用しております。ソースコードは GitHub 上にて公開中ですので、ご興味のある方はリポジトリも覗いてみてください。
